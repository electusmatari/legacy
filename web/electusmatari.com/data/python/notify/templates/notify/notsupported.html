Operation Notifications

ElectusMatari.com supported the notification of upcoming operations using the JavaScript Notifications API. This is the same API that is used by Google Mail for on-desktop notifications.

This means that you will get on-screen announcements of upcoming operations as long as you have any page on the EM forums open in a browser window somewhere.

You are likely reading this because your browser does not support this API by default. Only Google Chrome does by default. Other browsers need to install plugins.

  • HTML Desktop Notifications Plugin for Firefox
  • Chrome supports Web Notifications by default
  • Opera does not as of yet support this
