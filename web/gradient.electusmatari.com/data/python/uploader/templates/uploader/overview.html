Gradient Data Uploader

The Gradient Data Uploader is a client program that monitors your EVE client's cache files for interesting game data, and uploads this data to the server. This information includes for example the market orders you are currently viewing. The client is fully customizeable so you have full control over which information you share and which you don't.

And yes—this is absolutely legal according to CCP.

Uploading Data in 3 Simple Steps

Step 1: Install the Uploader

Download the installer and execute it. It will automatically install itself on your computer.

Step 2: Get Your Authentication Token

To minimize abuse, we need to associate every upload with your forum account. To allow GDU to identify the uploads as coming from you, it needs to know an authentication token. Simply copy it into the text field of the uploader.

Step 3: Upload Away!

Once the uploader is configured with your authentication token and running, you just play the game - it will do its work in the background. You can even configure it to auto-start in the configuration panel, so you don't even have to remember to start it before you play EVE. It should not interfere with other games or programs in any way.

If you would like to spend some dedicated time uploading data, you can visit our auto-upload page with the In-Game Browser. The page will use the In-Game Browser's JavaScript methods to open market windows etc. to check for information. And if you "trust" the auto-uploader page with your In-Game Browser, it will be able to dedicatedly figure out what it needs to see.
